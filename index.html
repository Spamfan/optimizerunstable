How To Use / FAQ▼
1) Use WARP to download an inventory print for each carrier.▼
How to find a carrier's "print inventory" screen on WARP▼
  • Log into warp as normal (use + new tab button on chrome)
  • Go to Inventory Management > View Available Inventory.
  • Select a postpaid carrier (AT&T, Verizon Wireless, or T-Mobile). You may need to use the top bar < > buttons to find it first.
  • Make sure the “Instock” box is checked, then click More Options () and choose Print.
How to save an inventory print as a PDF▼
  • Change the destination printer by clicking your stores printer (ie “CKPR1”) and changing it to “Save as PDF”, then click save.
        If you modified printer settings like scale and margin, reset these settings to their defaults before saving.
  • Name the file after its carrier (ie ATT.pdf, VZW.pdf, or TMO.pdf). If you've done this in the past, make sure to overwrite your previous save(s).
  • The above steps (navigating to the print screen then saving) should be done for each carrier whenever you want to print inventory.
2) On Optimizer, click "upload files," select the three PDFs you made, then click print.▼
I can't find the inventory files I'm trying to upload.▼
This happens when WARP files are saved in one directory while Optimizer tries to find them somewhere else.
The recommended and quickest solution: drag and drop each file from the WARP save window into optimizer.▼
  • Keep WARP open in a separate window. If helpful, rearrange the WARP and Optimizer windows so you can see both at once.
  • In the WARP window, after saving all needed carriers, start to save any one of the carriers a second time so that all carrier PDFs you need are visible in the save dialogue.
  • With the save dialogue open from WARP, click and drag each file from the warp save dialogue box into the upload files box on optimizer.
Longer term solution: tell optimizer and WARP to save PDFs to the same location (experimental)▼
Note: these instructions are a work in progress and may not work on every Walmart computer. To try and To reassign Optimizer's open dialogue directory:
  • click upload files
  • type Desktop\testABC.pdf into the file name field and click open.
  • If prompted “this file does not exist” and / or “your administrator has blocked this action,” press okay.
  • If the open dialogue doesn't close by itself, manually close it by clicking its x on the top right of the window.
  • Click upload files again, optimizer should now open to the correct file path every time.
NOTE: for this to work, the text preceding \testABC.pdf must be the same directory the carrier files are saving to. To check where they are saving, save a carrier PDF from warp and see what directory it lands in by reading it's directory above. If it isn't desktop, you can copy the directory by clicking on the file address bar and using Ctrl + c, then paste the directory into the step above.
Example: if carrier PDFs are saving to mpt user mpt user instead of desktop, copy that directory then put it at the beginning of the step 2 directory above, so it shows as mpt user mpt user\testABC.pdf, then continue directions as normal.
How to upload multiple PDFs at once▼
  • If carrier files are visible in the Optimizer open dialogue, you can upload them all simultaneously by using Ctrl + left click on each file then clicking open, or by dragging left click across all three then clicking open.
Why does my output document only say save and not print?▼
  • This happens if the user forgets to change the destination printer back from “Save as PDF” to your store's printer, such as “CKPR1.” Simply re-select your stores printer from that drop down menu then click print.
3) Optional: customize the list by marking devices as unavailable, adding comments, and more.▼
Hide or highlight devices with the customize entries button.▼
Click the “Customize Entries” button to view all detected devices.
You can highlight phones slightly or partially to emphasize deals you think are better that day.▼
  • Partial Highlight (left checkbox): Highlights only the phone's name. (Recommended for good deals).
  • Full Highlight (right checkbox): Highlights the entire row for that device. (Recommended for extra good deals).
  • Master Checkboxes: Use the main checkboxes at the top of the list to toggle partial or full highlights for all inventory at once.
Unsellable devices can be hidden from the output list by using the hide buttons.▼
  • Hide (toggle): Use this button to hide/unhide individual listings
  • Use the unhide all button at the top to unhide all phones that were manually hidden.
Click Done to save your customizations. The Walmart computer remembers the hide and highlight preferences until they're changed again. You can always change highlight or hide settings by going to this screen again.
Change formatting options like showing likely EDLPs or isolating Apple devices.▼
Toggle likely EDLPs▼
  • Likely EDLPs can be hidden from the output document to see fuller device names.
  • EDLP data waits for a human to notice an installment change and manually post it to the optimizer database, so it can sometimes be inaccurate or outdated. ☁️✅ means the database is working and showing the latest reported prices, while ☁️❌ means Optimizer can't see the database, and is using a static price backup, usually only a month or so old.
  • “Likely EDLPs” are usually accurate but aren't guaranteed, always check installment price on ESP before performing a sale.
Isolating Apple devices▼
  • When isolate Apple devices is set to on, Apple devices will only appear once in their own category on the inventory report, rather than repeating for each carrier.
Add comments to your printout.▼
  • Comments are visible near the top of the output PDF. This can be used for something fun or helpful you want others to know that day such as “We've got this team!” or “the stylus is free on T-Mobile right now,” etc.
Troubleshooting / help▼
Experiencing issues? Independently try the solutions below from top to bottom. You can also reach out to the developer directly from the "About" section below.
1) Read the tutorial above to make sure the step you're struggling with is being performed correctly.
2) Make sure the Walmart Computer is properly rebooted at the beginning of each shift▼
  • To reboot the Walmart computer, close all windows, left click on the wallpaper, then press Alt + F4. Choose "restart" from the drop-down and press "OK."
  • Be aware that poorly maintained devices could take longer to reboot; performing this step often will help prevent this.
3) Make sure files are saved correctly▼
  • Avoid using periods in file names. Try to label each file according to their carrier, just as ATT.pdf, VZW.pdf, and TMO.pdf. (Or att, VZW, and TMO). A label such as 7.1.26att.pdf could confuse the computer since it has too many periods.
  • There should only be 3 WARP files saved at any given time! Walmart computers aren't designed to store inventory files. Please help maintain the computer by labelling each carrier identically to the day before. This will overwrite old save information so the computer doesn't get bogged down.
4) If issues still persist, reset the optimizer cache.▼
  • Tap "show more settings” and turn "show developer options" on.
  • In the developer box appearing on the right, click "Clear optimizer cache"
  • HEADS UP, doing this will delete your saved optimizer settings: manually hidden devices and highlight preferences will both be reset.
  • To finish clearing the cache, refresh your browser page. (Ctrl + r)
