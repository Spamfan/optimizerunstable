HOW TO USE ▼
● Step 1: log into WARP and save each carrier's print as a PDF.
Show me how.
  1. Log into warp as normal (use + new tab button on chrome)
  2. Go to Inventory Management > View Available Inventory.
  3. Select a postpaid carrier (AT&T, Verizon Wireless, or T-Mobile) by clicking on it from the top navigation bar. You may need to use the top bar < > buttons to find it first.
  4. Make sure the “Instock” box is checked, then click More Options and choose Print.
  5. Change the destination printer by clicking “CKPR1” and changing it to “Save as PDF”, then click save.
          If you modified printer settings like scale and margin, please reset both to their defaults to ensure the files can save correctly.
  6. Name the file after its carrier as ATT.pdf, VZW.pdf, or TMO.pdf. If you've done this before, go ahead and do it again to overwrite the old save file(s) with fresh data.
  7. For each carrier, repeat steps 1-7 above until you have all three PDFs.
● Step 2: upload each carrier's PDF into optimizer.
Show me how.
  • Click upload files then select each carrier file (att.pdf, vzw.pdf, and tmo.pdf) and click open.
          💡 TIP: You can select multiple files at once by using Ctrl+left click or by dragging left click onto the files you'd like while in the open dialogue. You can also open each one individually.
  • NEW Carrier files not showing in the Open dialogue?
    This happens when Optimizer tries to open one directory (such as mpt user mpt user) instead of the directory where the carrier files are saved (usually Desktop). You can reassign the open dialogue to look in the right spot to show the files, optimizer will remember the correct directory.

    To reassign the open dialogue directory:
    1. click upload files
    2. type Desktop\testABC.pdf into the file name field and press or open.
    3. If prompted “this file does not exist” and / or “your administrator has blocked this action,” press okay.
    4. If the open dialogue doesn't close by itself, manually close it by tapping its x on the top right of the window.
    5. Click upload files again, optimizer should now open to the correct path every time.
    NOTE: for this to work, the text preceding \testABC.pdf must be the same directory the carrier files are saving to. To check where they are saving, save a carrier PDF from warp and see what directory it lands in by reading it's directory above. If it isn't desktop, you can copy the directory by clicking on the file address bar and using Ctrl + c, then paste the directory into the step above.

    Example: if carrier PDFs are saving to mpt user mpt user instead of desktop, copy that directory then put it at the beginning of the step 2 directory above, so it shows as mpt user mpt user\testABC.pdf, then continue directions as normal.
  • If preferred, you can instead drag and drop files into optimizer from the save dialogue on warp if WARP is open in a seperate window.
    Show me how.
    To do this, keep WARP open in a separate window and begin to save a fourth (any carrier) pdf after saving the regular 3. Change window sizes so you can see both at once if easier. With the save dialogue open from warp, click and drag each file from the warp save dialogue into the upload files box on optimizer.
● Step 3: (optional) customize the list by hiding inaccurate listings
Show me how.
  • Hide ( toggle): Use this to hide unwanted listings, such as demo-only or unlocked devices That are not sellable.
  • Use the unhide all button near the top to unhide all phones that were manually hidden.
  • Click Done to save your customizations. The Walmart computer remembers which phones you hide, so they'll manually be hidden again the next day. You can always unhide a phone by returning to this screen.
● Step 4: (optional) change settings like “show EDLPs” and “isolate iPhones”
More info
  • EDLP data waits for a human to notice an installment change and manually post it to the optimizer database, so it can sometimes be inaccurate or outdated. ☁️✅ means the database is working and showing the latest reported prices, while ☁️❌ means Optimizer can't see the database, and is using a static price backup, usually only a month or so old. “Likely EDLPs” are usually accurate but aren't guaranteed, always check installment price on ESP before performing a sale.
  • When isolate Apple products is turned on, Apple devices will only appear once in their own category on the inventory report, rather than repeating for each carrier.
● Step 5: (optional) customize the list by highlighting listings you think deserve more attention
More info
  • Partial Highlight (left checkbox): Highlights only the phone's name. (Recommended for good deals).
  • Full Highlight (right checkbox): Highlights the entire row for that device. (Recommended for extra good deals).
  • Master Checkboxes: Use the main checkboxes at the top of the list to toggle partial or full highlights for all inventory at once.
● Step 6: (optional) add comments to your printout for others to read.
More info
  • Comments are visible near the top of the printout. This can be used for something fun or helpful you want others to know that day such as “We've got this team!” or “the stylus is free on T-Mobile right now,” etc.
● Step 7: print the combined inventory list
Show me how.
  • Click the blue "Preview or Print" button.
  • A preview pane showing the combined PDF will appear. Right-click anywhere on the PDF preview and choose Print.
  • Important: Be sure to re-select your store's printer (e.g., "CPR1") as the destination so you print the combined file instead of saving it as a PDF again.
